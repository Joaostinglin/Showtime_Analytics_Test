
## Showtime Analytics Test

This is a test for Showtime Analytics Company

You can check that live on [this link](https://showtimeanalytics.herokuapp.com/).

## Pre-Requirements

* [NodeJs](https://nodejs.org/en/)
* Npm (It will be installed when NodeJs installing)



## Setup the Project

```
git clone https://github.com/Joaostinglin/Showtime_Analytics_Test.git
cd Showtime_Analytics_Test
Npm i (To install all dependencies)
```


## Run Project

Inside the project folder, run the command:
```
Npm start
```
It will open a browser, running the appication.
